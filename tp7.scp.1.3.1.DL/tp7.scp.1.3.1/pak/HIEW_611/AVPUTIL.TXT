Загрузка файла осуществляется либо через Object -> Load file, либо путем вызова
через командную строку:
avputil.com myfile.bin (можно загружать отдельные сектора).

Выход: alt+X, F10.

Управление с клавиатуры:

F1 -трассировка программы на одну инструкцию;

F2 -шаг.
По смещению следующей инструкции временно ставится 0cch, отладчик перехватывает
int 03 и ожидает перехода на себя -удобно для обхода конструкций вида loop,
int xx, call. Не использовать для отладки содерж. ROM.

alt+F1 -анимированный трейс;
alt+F2 -анимированный шаг;
можно выбирать delay, или задержку перед след. инструкцией.

cltrl+F1 -запустить код с текущего смещения;

F5 -развернуть текущее окно до полного экрана (только 80x25).
К сожалению, AVPutil не поддерживает режим 80x50, т.к. вывод осуществляется
напрямую в видеопамять.

alt+F5 -user screen (показать экран, находящийся "под" окнами отладчика).

F6 -смена активного окна.
По умолчанию состояния регистров, это дизасм и данные в hex -представлении.

Резидентный модуль: Setup -> Stay resident.

Перед этим можно перехватить одно // несколько прерываний, сделав это через
alt+I, затем, нажав на нужное прерывание, в открывшемся окне выбрать
Interceptor. При вызове перехваченного прерывания появляется окно с указанием
PSP модуля в памяти, который его вызвал, отображается сост. регистров и запрос
на действие пользователя. Free -отцепиться от инта. Trace -всплыть в
отладчик. Имеется буфер на несколько вызовов, по которому можно перемещаться
клавишами упр. курсором.

Доп. возможности:

Особенно приятно, что AVPutil позвляет трейсить прерывания (alt+I, выбрать инт,
enter, выбрать Trace и указать значения регистров при вызове прерывания).
